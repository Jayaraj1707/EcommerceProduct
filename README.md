
## Technologies Used
- Java 17
- Selenium WebDriver
- TestNG
- Log4j
- ExtentReports

## How to Run
1. Clone the repo
2. Run `mvn clean test` or execute `testng.xml` from your IDE
3. View results in `/reports/ExtentReport.html`

## Features
- Page Object Model
- Detailed logs
- Screenshots on failure
- Extent HTML Reports
